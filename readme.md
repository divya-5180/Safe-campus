# SafeCampus - Disaster Preparedness System

A web application that helps students quickly learn emergency response procedures through scenario-based guidance, safety checklists, and quick-access emergency resources.

## Features
- 4 Disaster Types: Earthquake, Fire, Flood, Cyclone
- Step-by-step emergency response actions
- Preparedness checklist with scoring
- Emergency contacts with Call and SMS
- BFS Evacuation Algorithm
- JSON Schema output

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python main.py
3. Open browser and go to:
   http://127.0.0.1:5000
